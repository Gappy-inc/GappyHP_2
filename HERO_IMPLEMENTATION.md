# ヒーローセクション実装ガイド

画像のような左右分割のヒーローセクションを実装する方法を説明します。

## 主な実装方法

### 1. SVGパスを使った方法（推奨）

波のような有機的な境界線を作る最も柔軟な方法です。

#### 構造
```
<section>
  ├── 左側: グラデーション背景 + コンテンツ
  │   └── SVG波形状（絶対配置、右端）
  └── 右側: 画像
```

#### ポイント
- SVGの`path`要素で波形状を描画
- `preserveAspectRatio="none"`で縦方向に引き伸ばし
- 複数の波レイヤーを重ねて有機的な形状に

### 2. CSS clip-pathを使った方法

シンプルですが、複雑な形状には制限があります。

```css
.clip-wave {
  clip-path: polygon(
    0% 0%,
    85% 0%,
    90% 20%,
    85% 40%,
    90% 60%,
    85% 80%,
    90% 100%,
    0% 100%
  );
}
```

### 3. 疑似要素を使った方法

CSSの`::before`や`::after`で波形状を作成。

## 実装の詳細

### SVGパスで有機的な波を作る

SVGパスは以下の要素で構成：
- `M`: 移動（Move to）
- `C`: 3次ベジェ曲線（Curve to）
- `Q`: 2次ベジェ曲線（Quadratic）
- `T`: スムーズな曲線の継続

### グラデーション背景

Tailwind CSSの`bg-gradient-to-br`を使用：
- `from-teal-500`: ティール色
- `via-teal-300`: 中間色
- `to-green-50`: 薄い緑

### レスポンシブ対応

- モバイル: 縦並び（左側のみ表示）
- デスクトップ: 横並び（左右分割）

## 改善のポイント

1. **波形状の調整**: SVGパスの座標を調整して理想の形状に
2. **パフォーマンス**: SVGをインライン化してHTTPリクエストを削減
3. **アクセシビリティ**: 適切なalt属性とARIAラベル
